텍스트 입력 필드는 사용자가 키보드로 글자, 숫자, 기호 등이 조합된 한 줄의 짧은 텍스트를 입력하는 경우에 사용하는 요소이다.

## 사용 지침

이름·아이디·검색어 같은 짧은 입력에 사용하고, 라벨·플레이스홀더·오류 메시지·아이콘을 일관된 위치에 배치해 폼 흐름을 예측 가능하게 만듭니다.
