텍스트 영역은 사용자가 키보드로 글자, 숫자, 기호 등이 조합된 여러 줄의 텍스트를 입력하는 경우에 사용하는 요소이다.

## 사용 지침

민원 사유·의견·후기처럼 한 줄로 표현하기 어려운 입력에 사용하고, 글자 수 카운터·최대 길이·크기 조절 정책을 명시해 사용자에게 입력 한계를 알려 줍니다.
