## 사용 지침

본문 흐름이 끝난 뒤 사용자가 추가 정보나 약관·개인정보처리방침에 접근할 수 있도록 항상 페이지 마지막에 배치하고, 화면 너비에 따라 컬럼 수를 조정합니다.
