코치마크는 사용자에게 새로 도입된 기능을 안내하거나, 여러 단계를 거쳐 수행해야 하는 복잡한 과업을 사용자가 보다 쉽게 완료할 수 있도록 세부 수행 단계별로 고맥락적 도움말을 제공하는 컴포넌트이다.

## 사용 지침

신규 기능 출시 직후 핵심 UI 위치를 알려야 할 때 한정적으로 사용하고, 같은 사용자에게 반복 노출하지 않도록 표시 여부를 저장합니다.
