셀렉트는 사용자에게 여러 개의 옵션 목록을 팝업으로 제공하여 그 중 한 개의 값을 선택할 수 있도록 하는 경우에 사용한다.

## 사용 지침

옵션 수가 많아 라디오로 펼치기 부담스러울 때 사용하고, 옵션 수가 적고 항상 노출이 더 효율적인 경우에는 라디오 버튼 사용을 우선 검토합니다.
