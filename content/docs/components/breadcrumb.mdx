브레드크럼을 통해 사용자는 탐색 중인 화면의 상위 수준 화면으로 이동할 수 있다.

## 사용 지침

본문 시작 바로 위에 두어 상위 단계로 빠르게 돌아갈 수 있게 하며, 검색·딥링크로 진입한 사용자가 사이트 구조를 빠르게 파악할 수 있도록 도와줍니다.
