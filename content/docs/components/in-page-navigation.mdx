## 사용 지침

본문이 여러 H2 단락으로 구성된 안내 페이지에서 우측 또는 상단에 배치하고, 스크롤 위치에 따라 현재 단락을 강조해 사용자가 페이지 구조를 한눈에 파악하도록 합니다.
