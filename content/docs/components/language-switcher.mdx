언어 변경은 서비스의 콘텐츠를 표시할 언어를 변경하거나 별도의 외국어 서비스로 이동하는 데 사용되는 요소이다.

## 사용 지침

다국어를 지원하는 누리집의 헤더 우측 유틸리티 영역에 배치하고, 현재 선택된 언어를 강조하면서 lang 속성을 함께 갱신해 스크린 리더가 언어 전환을 정확히 인식하도록 합니다.
