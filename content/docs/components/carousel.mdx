캐러셀은 하나의 콘텐츠 영역 내에 여러 개의 서로 다른 콘텐츠를 표시할 수 있는 컴포넌트이다.

## 사용 지침

주요 정책·홍보 콘텐츠 등 동등한 가치의 항목 3~5개를 메인 영역에 배치할 때 사용하고, 자동재생을 사용할 경우 일시정지 컨트롤과 키보드 이동을 반드시 제공합니다.
