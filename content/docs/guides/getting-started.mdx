description: 빈 Next.js 프로젝트에서 첫 KRDS 버튼을 띄우기까지, 처음부터 끝까지 따라 해봅니다.
---

이 가이드를 끝내면 Next.js 프로젝트에 KRDS 디자인 토큰이 설정되고, 화면에 KRDS 버튼이 렌더링됩니다. 5분이면 충분합니다.

## 준비물

- Node.js 20 이상
- Next.js(App Router) + Tailwind CSS v4 프로젝트

새 프로젝트부터 시작한다면:

```bash
pnpm create next-app@latest my-krds-app --typescript --tailwind --app
cd my-krds-app
```

<Steps>

<Step>

### shadcn CLI 초기화

krdscn/ui는 npm 패키지가 아니라 **컴포넌트 소스를 프로젝트로 복사하는 레지스트리**입니다. 먼저 shadcn CLI를 초기화합니다.

```bash
pnpm dlx shadcn@latest init
```

</Step>

<Step>

### 첫 컴포넌트 설치

Button 하나만 설치해봅니다.

```bash
pnpm dlx shadcn@latest add https://krdscn-ui.smlee.info/r/button.json
```

이 한 줄이 세 가지를 가져옵니다:

- `components/ui/krds/(action)/button.tsx` — Button 컴포넌트 소스
- `lib/cn.ts` — 클래스 병합 유틸리티
- `app/krds.css` — **KRDS 디자인 토큰 전체** (색상·타이포그래피·포커스 링)

토큰 스타일시트(`krds-theme`)는 모든 컴포넌트의 의존성으로 등록돼 있어서, 어떤 컴포넌트를 설치해도 자동으로 따라옵니다. 따로 챙길 필요가 없습니다.

</Step>

<Step>

### 토큰 스타일시트 연결

전역 CSS(`app/globals.css`)에 한 줄 추가합니다.

```css title="app/globals.css"
@import "tailwindcss";
@import "./krds.css";
```

이제 `bg-krds-primary-50`, `text-krds-body-md` 같은 KRDS 유틸리티 클래스를 프로젝트 어디서나 쓸 수 있습니다.

</Step>

<Step>

### 버튼 렌더링

```tsx title="app/page.tsx"
import { Button } from "@/components/ui/krds/(action)/button"

export default function Page() {
  return (
    <main className="flex min-h-svh items-center justify-center gap-3">
      <Button>신청하기</Button>
      <Button variant="secondary">임시저장</Button>
      <Button variant="tertiary">취소</Button>
    </main>
  )
}
```

개발 서버를 켜면 KRDS 파랑(`#256ef4`)의 버튼이 보입니다. Tab 키를 눌러보세요 — KRDS 공식 사양인 이중 포커스 링까지 이미 적용돼 있습니다.

```bash
pnpm dev
```

</Step>

</Steps>

## 폰트 (권장)

KRDS 공식 서체는 **Pretendard GOV**입니다. 토큰은 폰트 크기·행간만 관리하고 서체는 앱에서 설정합니다. [Pretendard GOV](https://github.com/orioncactus/pretendard/tree/main/packages/pretendard-gov)를 받아 `next/font`로 연결하세요.

```tsx title="app/layout.tsx"
import localFont from "next/font/local"

const pretendardGov = localFont({
  src: "./fonts/PretendardGOVVariable.woff2",
  variable: "--font-pretendard-gov",
})
```

```css title="app/globals.css"
@theme inline {
  --font-sans: var(--font-pretendard-gov), "Pretendard GOV", sans-serif;
}
```

## 소스가 내 것이 된다는 것

설치된 `button.tsx`는 여러분 프로젝트의 파일입니다. 패키지 업데이트로 덮어써질 걱정 없이 열어서 읽고, 서비스에 맞게 수정하면 됩니다. variant를 추가하고 싶으면 cva 정의에 한 줄 더하면 끝입니다.

전체 컴포넌트(41개)를 한 번에 가져오려면:

```bash
pnpm dlx shadcn@latest add https://krdscn-ui.smlee.info/r/krds-all.json
```
