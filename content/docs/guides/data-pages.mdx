지원 사업 공고 페이지를 예시로 데이터 표시 컴포넌트를 조합합니다.

## 설치

```bash
pnpm dlx shadcn@latest add \
  https://krdscn-ui.smlee.info/r/structured-list.json \
  https://krdscn-ui.smlee.info/r/table.json \
  https://krdscn-ui.smlee.info/r/badge.json \
  https://krdscn-ui.smlee.info/r/critical-alert.json
```

## CriticalAlert

페이지 최상단에 배치하는 긴급 공지입니다. `type`으로 성격을 구분합니다 (`emergency | safety | info`).

```tsx
import { CriticalAlert, CriticalAlertAction, CriticalAlertMessage } from "@/components/ui/krds/(layout)/critical-alert"

export function NoticeAlert() {
  return (
    <CriticalAlert type="info">
      <CriticalAlertMessage>2026년 하반기 접수 일정이 변경되었습니다</CriticalAlertMessage>
      <CriticalAlertAction href="/notice/42">자세히보기</CriticalAlertAction>
    </CriticalAlert>
  )
}
```

## StructuredList

구조화 목록은 제목·설명·기간·메타데이터·액션으로 이루어진 목록 패턴입니다. 부분별 컴포넌트를 필요한 만큼 조합합니다.

```tsx
import {
  StructuredList,
  StructuredListBadge,
  StructuredListBody,
  StructuredListDescription,
  StructuredListGroup,
  StructuredListHeader,
  StructuredListPeriod,
  StructuredListTitle,
} from "@/components/ui/krds/(layout)/structured-list"

export function ProgramList() {
  return (
    <StructuredListGroup>
      <StructuredList variant="vertical">
        <StructuredListBody>
          <StructuredListHeader>
            <StructuredListBadge>복지</StructuredListBadge>
            <StructuredListTitle>아동수당 신청 안내</StructuredListTitle>
          </StructuredListHeader>
          <StructuredListDescription>만 8세 미만 아동에게 월 10만 원을 지급하는 제도입니다.</StructuredListDescription>
          <StructuredListPeriod>2026.01.01 — 2026.12.31</StructuredListPeriod>
        </StructuredListBody>
      </StructuredList>
    </StructuredListGroup>
  )
}
```

`StructuredListBadge`는 카테고리(복지, 교육), `StructuredListTag`는 해시태그(#디지털교육) 용도로 역할이 구분되어 있습니다.

## Table

KRDS 표는 `TableHead`(th)를 열 제목뿐 아니라 행 제목에도 사용합니다. 스크린리더가 셀의 맥락을 읽을 수 있게 하는 접근성 장치입니다.

```tsx
import { Table, TableBody, TableCell, TableHead, TableHeader, TableRow } from "@/components/ui/krds/(layout)/table"

const ROWS = [
  { id: "2026-0711-001", name: "김민준", status: "접수 완료" },
  { id: "2026-0711-002", name: "이서연", status: "검토 중" },
]

export function ApplicationTable() {
  return (
    <Table className="w-full">
      <TableHeader>
        <TableRow>
          <TableHead>접수번호</TableHead>
          <TableHead>신청자</TableHead>
          <TableHead>상태</TableHead>
        </TableRow>
      </TableHeader>
      <TableBody>
        {ROWS.map((row) => (
          <TableRow key={row.id}>
            <TableCell className="font-mono text-xs">{row.id}</TableCell>
            <TableCell>{row.name}</TableCell>
            <TableCell>{row.status}</TableCell>
          </TableRow>
        ))}
      </TableBody>
    </Table>
  )
}
```

<Callout>
  스크롤 영역에서 헤더를 고정할 때는 `bg-white` 같은 고정 색 대신 `sticky top-0 bg-krds-surface`처럼 시맨틱 토큰을
  사용해야 다크 모드에서 색이 유지됩니다.
</Callout>

## Badge

Badge는 외형(`type`: solid/outline/pastel)과 색(`variant`: 8종)을 조합합니다. 표의 상태 열에 문자열 대신 사용하면 시인성이 좋아집니다.

```tsx
import { Badge } from "@/components/ui/krds/(layout)/badge"

export function StatusBadges() {
  return (
    <div className="flex gap-2">
      <Badge type="pastel" variant="success">
        접수 완료
      </Badge>
      <Badge type="pastel" variant="warning">
        검토 중
      </Badge>
      <Badge type="outline" variant="secondary">
        마감
      </Badge>
    </div>
  )
}
```
