컴포넌트 각각의 API는 [컴포넌트 문서](/docs/components)에서 다룹니다. 가이드는 화면 단위 시나리오에서 어떤 컴포넌트를 어떤 순서로 조합하는지 안내합니다.

## 목차

| 가이드                                   | 내용                                                 |
| ---------------------------------------- | ---------------------------------------------------- |
| [시작하기](/docs/guides/getting-started) | 프로젝트 설정부터 첫 KRDS 컴포넌트 렌더링까지        |
| [페이지 셸](/docs/guides/page-shell)     | 공식 배너·헤더·푸터·운영기관 식별자로 기본 골격 구성 |
| [신청 폼](/docs/guides/building-a-form)  | 입력·선택·날짜 컴포넌트와 오류 메시지 처리           |
| [내비게이션](/docs/guides/navigation)    | 메인 메뉴·브레드크럼·사이드 메뉴·페이지네이션        |
| [목록과 표](/docs/guides/data-pages)     | 구조화 목록·표·배지·긴급 공지로 데이터 화면 구성     |
| [다크 모드](/docs/guides/dark-mode)      | 고대비 모드 적용과 색상 토큰 사용 규칙               |
